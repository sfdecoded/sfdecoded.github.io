Salesforce Decoded

Salesforce Decoded is a collection of tools, guides, and resources built to help Salesforce developers work faster and write better code.

The platform provides utilities for Apex, SOQL, debugging, and developer productivity.

Live website:
https://sfdecoded.github.io

---

Overview

Salesforce Decoded aims to become a central hub for Salesforce development resources. It combines practical developer tools with guides and troubleshooting references that help engineers solve real problems while building on the Salesforce platform.

The goal is to provide fast, useful, and accessible tools that improve everyday development workflows.

---

Features

Developer Tools

Utilities designed to simplify common Salesforce development tasks.

- SOQL Formatter
- Salesforce ID Converter
- JSON to Apex Converter
- Apex Code Generator
- Salesforce Icon Explorer
- Field Dependency Viewer

These tools help developers format queries, generate Apex structures, inspect metadata, and work more efficiently with Salesforce data.

---

Error Reference

Salesforce errors can be difficult to diagnose. Salesforce Decoded provides explanations and solutions for common platform errors including:

- Too Many SOQL Queries (101)
- Apex CPU Time Limit Exceeded
- Insufficient Access Cross Reference Entity
- Unable to Lock Row

Each page includes explanation, root cause, and suggested solutions.

Guides

The platform also includes development guides covering topics such as:

These guides focus on practical implementation and real-world scenarios.
